# Sustainability-Social-Value-Review
Power BI Social Value Impact analysis dashboard

This dashboard built for the impact  review analysis on the supplier works based on 5 categories:
1) Local Employment
2) Local SME
3) Decarbonisation
4) Apprenticeship
5) Volunteering
